# Design Brief: Catthode Theme

From CRT to OLED. Bringing warmth back to a world of cold themes.

## Visual Identity

Vibe: Retro-futuristic, High Contrast, Warm, Precise, Analog-meets-Digital.

Key Elements: subtle bloom/glow on text, crushed blacks, warm tungsten lighting.

## Color Palette (Hex Codes)

### Surface
Used for editor backgrounds and UI panels.

- base: #000000
- sidebar: #141414
- selection: #2d2d2d
- border: #636363

### Phosphor
Used for standard text and comments.

- text: #ffffff
- variable: #e8e8e8
- comment: #b3b3b3
- ignored: #757575

### Glow
The signature warm gradient used for main syntax highlighting.

- cat1 (Wheat): #fae2c8
- cat2 (Tan): #d9b98c 
- cat3 (Gold): #ffb86c 
- cat4 (Amber): #ff9e3b
- cat5 (Clay): #f08d49

### Accents
Used for specific syntax tokens (strings, errors, constants).

- red: #ff6b6b
- green: #b9d665
- cyan: #aee6d6
- blue: #9cd9e6
- purple: #eba4be

## Icon/Logo Inspiration Keywords
- Vacuum tube glowing in the dark.
- Abstract Flat geometric cat shape formed by neon amber lines.
- Retro terminal cursor block.
- Minimalist C glyph with a filament glow.
- Interference patterns on a black screen.
